# دليل أتمتة البريد الإلكتروني باستخدام OpenClaw وHimalaya

إدارة البريد الإلكتروني واحدة من أكثر المهام اليومية استهلاكًا للوقت. تخيل لو كان بإمكان مساعدك الذكي الشخصي قراءة رسائلك، تصنيفها، الرد على الرسائل الروتينية، وتنبيهك فقط بالأمور المهمة. مع **OpenClaw** ومهارة **Himalaya**، هذا ليس خيالًا - بل واقع يمكنك تطبيقه اليوم.

## ما هي مهارة Himalaya؟

Himalaya هي واجهة سطر أوامر (CLI) قوية لإدارة البريد الإلكتروني عبر بروتوكولات IMAP وSMTP. عند دمجها مع OpenClaw، تصبح أداة أتمتة ذكية قادرة على:

- قراءة وتحليل الرسائل الواردة
- كتابة وإرسال الردود
- البحث في البريد الإلكتروني
- تنظيم المجلدات والعلامات
- أرشفة وحذف الرسائل
- إدارة حسابات متعددة

كل هذا بقوة الذكاء الاصطناعي من Claude، مما يعني أن مساعدك يفهم السياق ويتخذ قرارات ذكية.

## التثبيت والإعداد

### المتطلبات الأساسية

قبل البدء، تأكد من تثبيت OpenClaw على نظامك. إذا لم تكن قد فعلت ذلك بعد، راجع [دليل تثبيت OpenClaw على Linux/Ubuntu](./2026-02-11-install-openclaw-linux-ubuntu.md).

### تثبيت Himalaya

```bash
# macOS
brew install himalaya

# Linux (Cargo)
cargo install himalaya

# أو من المصدر
git clone https://github.com/soywod/himalaya.git
cd himalaya
cargo install --path .
```

### إعداد الحساب

قم بإنشاء ملف تكوين Himalaya:

```bash
himalaya accounts configure
```

ستُسأل عن:
- عنوان البريد الإلكتروني
- خادم IMAP (مثل: imap.gmail.com)
- خادم SMTP (مثل: smtp.gmail.com)
- كلمة المرور أو رمز التطبيق

**ملاحظة مهمة:** لحسابات Gmail، ستحتاج إلى إنشاء "كلمة مرور التطبيق" من إعدادات الأمان في Google، وليس كلمة المرور العادية.

### التحقق من التثبيت

```bash
# اختبر بقراءة صندوق الوارد
himalaya list

# اقرأ أحدث رسالة
himalaya read 1
```

## سيناريوهات الأتمتة الذكية

الآن يأتي الجزء المثير - دمج Himalaya مع OpenClaw لإنشاء سيناريوهات أتمتة متقدمة.

### 1. مراقبة صندوق الوارد التلقائية

استخدم [وظائف Cron في OpenClaw](./2026-02-15-cron-jobs-automation-guide.md) لمراقبة البريد الوارد كل 30 دقيقة:

في ملف `HEARTBEAT.md` الخاص بك:

```markdown
## Email Check
- كل ساعة، تحقق من الرسائل الجديدة
- أبلغني بالرسائل العاجلة فقط
- صنف الباقي تلقائيًا
```

عندما يستلم OpenClaw نبضة القلب، سيقوم تلقائيًا بـ:

```bash
himalaya list --page-size 20 | grep "unread"
```

ثم يحلل الرسائل ويقرر:
- هل هي عاجلة؟ → إشعار فوري
- رسالة تسويقية؟ → أرشفة
- تحتاج رد؟ → صياغة رد ذكي

### 2. الرد التلقائي على الاستفسارات الروتينية

مع قوة Claude، يمكن لـOpenClaw فهم محتوى الرسائل والرد بذكاء:

**مثال:** شخص يسأل عن ساعات العمل:

```bash
# OpenClaw يقرأ الرسالة
himalaya read 42

# يفهم أنها سؤال عن ساعات العمل
# يصيغ ردًا مناسبًا
himalaya reply 42 --body "شكرًا لتواصلك. ساعات عملنا من 9 صباحًا حتى 5 مساءً من الأحد إلى الخميس."

# يرسل الرد
```

كل هذا يحدث تلقائيًا بدون تدخلك!

### 3. تصنيف وأرشفة ذكية

علّم OpenClaw كيفية تنظيم بريدك:

```bash
# إنشاء قواعد تصنيف
# الرسائل من "newsletter@*" → مجلد "Newsletters"
# الرسائل التي تحتوي "فاتورة" → مجلد "Invoices"
# الرسائل من الفريق → "Work"
```

OpenClaw يتعلم أنماطك ويطبقها تلقائيًا.

### 4. استخراج المعلومات الهامة

اطلب من OpenClaw استخراج معلومات محددة من رسائلك:

- "استخرج جميع أرقام الطلبات من رسائل Amazon هذا الشهر"
- "اجمع كل اجتماعات Zoom المرسلة هذا الأسبوع"
- "ابحث عن رسائل تحتوي على مرفقات PDF عن المشروع X"

```bash
himalaya search "from:amazon.com subject:order" --page-size 50
```

### 5. متابعة الرسائل المعلقة

علّم OpenClaw تتبع الرسائل التي تحتاج متابعة:

في ملف `MEMORY.md`:

```markdown
## Pending Emails
- 2026-02-10: انتظار رد من العميل ABC (تذكير بعد 3 أيام)
- 2026-02-12: متابعة عرض الأسعار مع شركة XYZ
```

OpenClaw سيتحقق تلقائيًا وينبهك عند حلول موعد المتابعة.

## حالة استخدام عملية: مدير مستقل

دعنا نرى كيف يستخدم "أحمد"، مطور مستقل، هذه الأتمتة:

### الصباح (8:00 ص)
```bash
# OpenClaw يفحص البريد تلقائيًا
# يجد 15 رسالة جديدة:
# - 8 رسائل تسويقية → أرشفة
# - 4 إشعارات GitHub → تصنيف "Development"
# - 2 استفسارات عملاء → تنبيه فوري
# - 1 فاتورة → حفظ في "Invoices" + إضافة للتقويم
```

**إشعار Telegram:**
"📧 رسالتان تحتاجان انتباهك:
1. عميل جديد يسأل عن توفرك (عاجل)
2. متابعة من مشروع ABC"

### أثناء اليوم
أحمد يركز على العمل، OpenClaw يدير البريد:
- يرد تلقائيًا على الأسئلة الشائعة
- يصنف الرسائل الجديدة
- يستخرج معلومات المشاريع ويحدث ملفات العمل

### المساء (6:00 م)
```bash

"📊 ملخص البريد اليوم:
- 32 رسالة معالجة
- 5 ردود تلقائية مُرسلة
- 3 رسائل تحتاج ردك الشخصي
- 2 فواتير جديدة (المجموع: 1,250 دولار)"
```

## نصائح للأمان والخصوصية

عند أتمتة البريد الإلكتروني، الأمان أولوية:

### 1. استخدم كلمات مرور التطبيقات
لا تستخدم كلمة المرور الرئيسية أبدًا. استخدم كلمات مرور مخصصة للتطبيقات يمكن إلغاؤها.

### 2. تشفير بيانات الاعتماد
احفظ بيانات Himalaya في مكان آمن:

```bash
chmod 600 ~/.config/himalaya/config.toml
```

### 3. راجع الإجراءات التلقائية
في البداية، اجعل OpenClaw يطلب تأكيدك قبل إرسال الردود:

في `AGENTS.md`:

```markdown
## Email Rules
- اقرأ وصنف تلقائيًا
- اقترح الردود لكن **انتظر موافقتي قبل الإرسال**
- أرشف التسويقية تلقائيًا (آمن)
```

### 4. حماية البيانات الحساسة
راجع [دليل الأمان والخصوصية في OpenClaw](./2026-02-14-openclaw-security-privacy-guide.md) لفهم كيفية حماية معلوماتك.

## الأوامر الأساسية لـHimalaya

إليك مرجع سريع للأوامر الأكثر استخدامًا:

```bash
# قراءة البريد
himalaya list                    # قائمة الرسائل
himalaya read <id>               # قراءة رسالة محددة
himalaya search "query"          # البحث

# كتابة وإرسال
himalaya write                   # رسالة جديدة
himalaya reply <id>              # الرد
himalaya forward <id>            # إعادة توجيه

# إدارة
himalaya move <id> <folder>      # نقل رسالة
himalaya delete <id>             # حذف
himalaya flags set <id> seen     # تعليم كمقروءة

# حسابات متعددة
himalaya -a work list            # حساب العمل
himalaya -a personal list        # حساب شخصي
```

## الأخطاء الشائعة وحلولها

### المشكلة: "Connection refused"
**الحل:** تحقق من إعدادات جدار الحماية والمنافذ (IMAP: 993, SMTP: 465/587)

### المشكلة: "Authentication failed"
**الحل:** استخدم كلمة مرور التطبيق، وليس كلمة المرور العادية (خاصة Gmail)

### المشكلة: Himalaya بطيء
**الحل:** استخدم `--page-size` لتحديد عدد الرسائل المحملة:
```bash
himalaya list --page-size 20
```

### المشكلة: OpenClaw لا يكتشف الرسائل الجديدة
**الحل:** تأكد من تكوين heartbeat في `HEARTBEAT.md` أو استخدم cron job مخصص

## الخلاصة

أتمتة البريد الإلكتروني مع OpenClaw وHimalaya تحولك من مجرد "إدارة البريد الوارد" إلى "التركيز على ما يهم". بدلاً من قضاء ساعات يوميًا في فرز الرسائل، دع مساعدك الذكي يتولى المهام الروتينية بينما تركز أنت على العمل الحقيقي.

هذا ليس مجرد توفير للوقت - إنه تحول في طريقة العمل. مع [OpenClaw كمساعد شخصي ذكي](./2026-02-13-create-personal-ai-assistant.md)، بريدك الإلكتروني يصبح أداة قوية تعمل لصالحك، وليس عبئًا يستهلك يومك.
