# تثبيت أوبن كلاو (OpenClaw) على Windows: الدليل الشامل خطوة بخطوة

**ما ستتعلمه في هذا المقال:** ستتعلم طريقتين لتثبيت أوبن كلاو (OpenClaw) على Windows: الطريقة المباشرة عبر Node.js للمستخدم العادي، وطريقة WSL2 للمطورين. سنغطي أيضًا حل المشاكل الشائعة وأفضل الممارسات.

هل تستخدم Windows وتريد الاستفادة من قوة **أوبن كلاو (OpenClaw)** كمساعد ذكاء اصطناعي شخصي؟ أنت في المكان الصحيح. يدعم أوبن كلاو (OpenClaw) أنظمة Windows بشكل كامل، وفي هذا الدليل سنأخذك خطوة بخطوة من الصفر حتى تشغيل مساعدك الذكي بنجاح.

---

## المتطلبات الأساسية

قبل البدء، تأكد من أن نظامك يستوفي المتطلبات التالية:

| المتطلب | الحد الأدنى |
|---------|------------|
| نظام التشغيل | Windows 10 (build 19041) أو Windows 11 |
| الذاكرة العشوائية | 4 GB RAM (8 GB مُوصى به) |
| مساحة القرص | 500 MB فارغة |
| الاتصال بالإنترنت | مطلوب للتثبيت وللـ API |

## الطريقة الأولى: التثبيت المباشر عبر Node.js (الأبسط)

### الخطوة 1: تثبيت Node.js

أوبن كلاو (OpenClaw) يعتمد على Node.js. إذا لم يكن مثبتًا لديك، اتبع الخطوات التالية:

1. انتقل إلى [nodejs.org](https://nodejs.org)
2. حمّل الإصدار **LTS** (الإصدار المستقر الموصى به)
3. شغّل ملف الإعداد `.msi` واتبع معالج التثبيت
4. اقبل الاتفاقية وانقر **Next** في كل خطوة

**تحقق من التثبيت:** افتح **Command Prompt** (ابحث عن `cmd` في قائمة Start) ونفّذ:

```bash
node --version
npm --version
```

يجب أن يظهر لك رقم إصدار كـ `v20.x.x` و `10.x.x`.

---

### الخطوة 2: تثبيت أوبن كلاو (OpenClaw)

في نفس نافذة Command Prompt، نفّذ الأمر التالي:

```bash
npm install -g openclaw
```

قد يستغرق هذا دقيقة أو دقيقتين حسب سرعة اتصالك. انتظر حتى ينتهي التثبيت.

**تحقق من النجاح:**

```bash
openclaw --version
```

إذا ظهر رقم الإصدار، فقد نجح التثبيت! 🎉

---

### الخطوة 3: الحصول على مفتاح Anthropic API

أوبن كلاو (OpenClaw) يعمل مع Claude من Anthropic. تحتاج إلى مفتاح API:

1. انتقل إلى [console.anthropic.com](https://console.anthropic.com)
2. أنشئ حسابًا مجانيًا (أو سجّل الدخول)
3. اذهب إلى **API Keys** وانقر **Create Key**
4. انسخ المفتاح واحفظه في مكان آمن, لن تراه مرة أخرى!

---

### الخطوة 4: الإعداد الأول

في Command Prompt، شغّل معالج الإعداد:

```bash
openclaw setup
```

سيطلب منك:
- **Anthropic API Key**: الصق المفتاح الذي حصلت عليه
- **اسم مساعدك**: اختر اسمًا تريده (مثلاً: `مساعدي`)
- **المجلد الافتراضي**: اضغط Enter لقبول القيمة الافتراضية

بعد الانتهاء، شغّل أوبن كلاو (OpenClaw):

```bash
openclaw start
```

## الطريقة الثانية: التثبيت عبر WSL2 (للمطورين)

إذا كنت مطورًا أو تريد بيئة Linux كاملة داخل Windows، فإن **WSL2** (Windows Subsystem for Linux) هو الخيار الأمثل. يوفر أداءً أفضل وتوافقًا أوسع مع أدوات Linux.

### الخطوة 1: تفعيل WSL2

افتح **PowerShell كمسؤول** (انقر بالزر الأيمن على Start → Windows PowerShell (Admin)) ونفّذ:

```powershell
wsl --install
```

هذا الأمر سيثبّت WSL2 وتوزيعة Ubuntu تلقائيًا. **أعد تشغيل جهازك** بعد اكتمال التثبيت.

### الخطوة 2: إعداد Ubuntu

بعد إعادة التشغيل، سيفتح Ubuntu تلقائيًا. أنشئ اسم مستخدم وكلمة مرور للنظام الفرعي، ثم حدّث الحزم:

```bash
sudo apt update && sudo apt upgrade -y
```

### الخطوة 3: تثبيت Node.js داخل WSL2

```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt install -y nodejs
```

### الخطوة 4: تثبيت أوبن كلاو (OpenClaw)

```bash
npm install -g openclaw
openclaw setup
```

اتبع نفس خطوات الإعداد كما في الطريقة الأولى.

## مشاكل شائعة وحلولها

### ❌ خطأ "npm is not recognized"

**السبب:** Node.js لم يُضَف إلى متغير PATH.

**الحل:**
1. أعد تشغيل Command Prompt بعد تثبيت Node.js
2. أو يدويًا: ابحث عن "Environment Variables" في Windows وأضف مسار Node.js

### ❌ خطأ "EACCES: permission denied"

**السبب:** صلاحيات التثبيت.

**الحل على Windows:** شغّل Command Prompt كمسؤول:
- ابحث عن `cmd`
- انقر بالزر الأيمن → **Run as administrator**
- ثم نفّذ أمر npm مجددًا

### ❌ أوبن كلاو (OpenClaw) لا يتصل بـ API

**السبب:** مفتاح API خاطئ أو شبكة محجوبة.

**الحل:**
```bash
openclaw config set ANTHROPIC_API_KEY=sk-ant-xxxxxx
```
تأكد من أن اتصالك بالإنترنت يعمل وأن المفتاح لم تنته صلاحيته.

### ❌ بطء الأداء في WSL2

**الحل:** تأكد من تخزين مشاريعك داخل نظام ملفات Linux (مسار `/home/username/`) وليس داخل `/mnt/c/`.

## الأسئلة الشائعة

### هل يعمل أوبن كلاو (OpenClaw) على Windows 10 القديم؟
يعمل أوبن كلاو (OpenClaw) على Windows 10 build 19041 أو أحدث. إذا كان إصدارك أقدم، يُنصح بتحديث Windows أولاً. للتحقق من إصدارك، اكتب `winver` في قائمة Start.

### أيهما أفضل: Node.js المباشر أم WSL2؟
للمستخدم العادي الذي يريد البدء سريعًا، Node.js المباشر هو الأفضل (5 دقائق للتثبيت). للمطورين الذين يستخدمون أدوات Linux أو يريدون أداءً أفضل، WSL2 هو الخيار الأمثل.

### كم تكلفة استخدام أوبن كلاو (OpenClaw) على Windows؟
أوبن كلاو (OpenClaw) مجاني تمامًا. التكلفة الوحيدة هي رسوم Anthropic API التي تبلغ حوالي 5-15$ شهريًا للاستخدام المعتدل. يمكنك البدء بـ 5$ رصيد مجاني من Anthropic لتجربة الخدمة.
